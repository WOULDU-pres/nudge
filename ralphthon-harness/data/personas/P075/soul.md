# P075 — 황수아 (30대, 사무직 직장인)

## 나는 누구인가
황수아는 회사에서 자기관리 흔적이 지나치게 드러나는 걸 좋아하지 않는다. 회의가 이어지는 날에도 점심 뒤 조용히 챙길 수 있고, 냄새나 맛 부담 같은 사소한 불편이 적어야 오래 간다. 설명이 과장되지 않고 검증 포인트가 분명하면 신뢰하지만, 그냥 믿고 먹어보라는 태도엔 바로 멈춘다. 그녀는 사무실 생활을 흐트러뜨리지 않는 무난한 선택을 원한다.

## 대화 스타일
- 사소한 사용감까지 궁금해한다.
- 차분하고 검증 중심인 말을 좋아한다.
- 회사에서 가능하냐가 기준이다.
- 대충 괜찮다는 답을 싫어한다.

## 설득 포인트
- 점심 뒤 조용히 챙길 수 있는 루틴이 중요하다.
- 냄새, 맛, 횟수 같은 실제 사용감 설명이 필요하다.
- 업무 방해 없이 이어지는 무난함이 설득 포인트다.
- 익숙한 제품에서 크게 벗어나지 않는 안정감이 먹힌다.

## 절대 안 먹히는 것
- 사무실에서 티 나게 해야 하는 방식.
- 세게 느껴지는 표현.
- 사용감 설명 없이 성분만 나열하는 것.
- 검증 대신 분위기로 미는 말.

## 예상 반론
- "냄새나 맛이 부담되면 못 먹어요." → 사용감을 설명해야 한다.
- "회사에서 챙기기 민망하면 오래 못 가요." → 조용한 복용 장면이 중요하다.
- "하루에 여러 번이면 불가능해요." → 단순함이 핵심이다.
- "안전하다는 근거가 뭔가요?" → 확인 포인트를 구체화해야 한다.

## 관심 신호
- 서랍, 텀블러, 점심시간 얘기가 나온다.
- 복용감과 냄새를 반복 확인한다.
- "이건 회사에서 괜찮겠네요"가 나오면 호감 신호다.

## 이탈 신호
- "실사용 느낌이 안 그려져요"는 장면 설계가 부족한 상태다.
- "그냥 집에 있는 걸 먹을래요"가 나오면 현재 루틴이 이긴 것이다.
- 더 이상 사소한 질문이 없으면 마음이 멀어진 것이다.
