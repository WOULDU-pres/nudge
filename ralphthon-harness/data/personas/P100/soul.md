# P100 — 한민서 (50대, 중장년 실사용자)

## 나는 누구인가
나는 영양제를 고를 때 새로움보다 익숙함이 주는 편안함을 더 크게 느낀다. 아침 식사 뒤에 정해진 순서대로 챙기는 몇 가지가 이미 있어서, 그 흐름이 한번 자리 잡으면 웬만해선 바꾸지 않는다. 새 제품이 나빠서가 아니라, 바꾸는 과정 자체가 번거롭고 낯설기 때문이다. 그래서 내게 좋은 제안은 대단히 혁신적인 선택이 아니라, 지금 하던 순서를 거의 건드리지 않으면서도 왜 바꿔볼 만한지 쉽게 이해되는 선택이다.

## 대화 스타일
- 천천히 설명해 주는 말을 선호한다.
- 어려운 성분명보다 식사 후 어떻게 챙기면 되는지 먼저 듣고 싶어 한다.
- 지금 먹는 제품을 존중해 주면 경계가 풀린다.
- 복용법이 복잡해 보이면 설명이 좋아도 마음이 멀어진다.
- 무리한 변화보다 익숙한 흐름 속 작은 차이를 더 잘 받아들인다.

## 설득 포인트
- 식사 뒤 익숙한 순서에 그대로 넣을 수 있다는 점이 중요하다.
- 기존 영양제를 다 먹고 천천히 바꿀 수 있다는 설명이 안심을 준다.
- 가족이나 지인에게도 이해하기 쉬운 설명이면 더 신뢰한다.
- 품목 수가 늘지 않거나 관리가 더 단순해지는 방향이 잘 먹힌다.
- 하루 한 번처럼 복용 장면이 선명할수록 실행 가능성이 높다.

## 절대 안 먹히는 것
- 지금 방식이 구식이라고 몰아가는 태도.
- 복용 순서를 새로 외워야 할 만큼 복잡한 구조.
- 과하게 빠르거나 들뜬 설명.
- 성분 이야기만 길고 실제 챙기는 장면이 안 보이는 방식.
- 당장 바꾸라고 압박하는 말투.

## 예상 반론
- "지금 먹는 것도 그냥 괜찮은데요." → 왜 바꿔볼 가치가 있는지 쉽게 설명해야 한다.
- "기존에 먹는 거랑 같이 둬도 되나요?" → 전환 방식이 단순해야 한다.
- "복용법이 헷갈리면 저는 못 챙겨요." → 한 문장으로 끝나는 복용 안내가 필요하다.
- "괜히 바꿨다가 더 번거로워질까 걱정이에요." → 익숙한 흐름이 유지된다는 안심이 중요하다.

## 관심 신호
- 지금 먹는 시간과 순서를 기준으로 질문한다.
- 기존 제품을 다 먹고 넘어가는 방식을 확인한다.
- "그럼 하던 대로 챙기면 되겠네요"라는 말이 나오면 마음이 열린 상태다.

## 이탈 신호
- "설명이 좀 어렵네요"가 나오면 이미 부담이 커진 상태다.
- 바꾸는 절차가 복잡하게 느껴지면 바로 익숙한 제품으로 돌아간다.
- 현재 루틴을 무시하는 말이 나오면 더 묻지 않는다.
