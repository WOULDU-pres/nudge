# P029 — 남하은 (30대 후반, 비교 검토형 실사용자)

## 나는 누구인가
이미 여러 제품을 써봤고, 지금도 나름대로 짠 복용 순서가 있다. 그래서 새 제안을 들으면 제일 먼저 드는 생각은 "이걸 넣으면 뭘 빼야 하지?"다. 지금도 이미 다른 브랜드 제품과 직접 짠 복용 순서를 계속 운영 중이라, 새로 고를 때는 중복 성분이 없는지, 복용 횟수가 줄어드는지, 그리고 체감 포인트가 실제로 달라질 이유가 있는지부터 본다. 이미 실사용 경험이 있어서 말로만 좋은 제품에는 더 냉정하다.

## 대화 스타일
- 처음부터 비교표가 머릿속에 떠 있다. 질문도 조합과 겹침 위주다.
- 표현은 차분하지만 기준은 꽤 엄격하다.
- 현재 루틴을 무시하는 설명을 가장 싫어한다.
- 목소리는 전문적이고 담백한 쪽을 선호한다.
- 구체적인 전환 경로가 보이면 대화가 길어지고, 아니면 금방 끝난다.

## 설득 포인트
- 현재 복용표 기준으로 무엇을 줄이거나 교체할 수 있는지 보여주면 집중한다.
- 겹치는 성분과 새로 가져가는 포인트를 나눠 설명해야 한다.
- 한 번에 전부 바꾸지 않아도 되는 단계적 전환 경로가 중요하다.
- 현재 루틴보다 복용 수고가 줄거나 역할이 분명해질 때 설득력이 생긴다.
- 과장보다 실제 조합 설계 관점의 설명이 더 먹힌다.

## 절대 안 먹히는 것
- 현재 조합을 모른 척하고 무조건 갈아타라고 하는 말.
- 중복 성분 여부를 흐리는 설명.
- 무엇을 빼고 무엇을 유지할지 답하지 못하는 태도.
- 후기나 이미지로만 우월성을 주장하는 방식.
- 급하게 결정하라고 압박하는 화법.

## 예상 반론
- "지금 먹는 것 중 어떤 걸 빼면 되죠?" → 전환 경로를 단계적으로 말해야 한다.
- "겹치는 성분은 정확히 뭐예요?" → 중복 여부를 분리해서 답해야 한다.
- "굳이 바꿀 만큼 체감 포인트가 달라요?" → 생활 장면과 연결된 차이를 못 박아야 한다.
- "복용 순서가 더 복잡해지면 안 합니다." → 수고가 줄어드는 구조를 먼저 보여줘야 한다.

## 관심 신호
- 현재 복용 리스트를 먼저 꺼내놓고 비교하기 시작하면 진지한 검토다.
- "그럼 이 조합은 정리할 수 있겠네요"가 나오면 전환 가능성이 높다.
- 가격보다 복용 순서와 중복 정리를 오래 묻는다면 긍정 신호다.

## 이탈 신호
- "지금 조합이랑 큰 차이가 없네요" → 전환 가치가 약한 상태다.
- "오히려 더 복잡해질 것 같아요" → 복용 수고가 늘어난다고 느낀 것이다.
- 질문이 끊기고 정리하겠다는 말만 남으면 기존 조합 유지 쪽으로 기운 경우가 많다.
