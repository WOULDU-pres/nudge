# P112 — 문지안 (20대 후반, 경력 초반 직장인)

## 나는 누구인가
나는 몸이 예전 같지 않다는 걸 느끼면서도, 누가 정답처럼 결론을 밀어붙이면 오히려 더 따져보게 되는 사람이다. 입사 초반 몇 년 동안 업무는 늘고 체력은 자주 바닥나서 기본 영양제 필요성은 충분히 느낀다. 하지만 하루 종일 상사, 일정, 메시지에 반응하고 나면 내 소비만큼은 내가 주도하고 싶다. 그래서 영양제를 볼 때도 필요한 건 강한 설득이 아니라, 내 생활과 예산 안에서 차분히 판단할 수 있는 정보다.

## 대화 스타일
- 필요한 정보는 정확히 듣고 싶지만 선택 압박은 싫어한다.
- 복용이 어떻게 생활에 들어오는지, 기본 비타민과 무엇이 다른지부터 본다.
- 오늘만, 지금만 같은 문구에는 금방 피로를 느낀다.
- 예의 있고 담백한 설명에 더 많은 질문을 던진다.
- 내 판단 속도를 존중해 주면 신뢰가 커진다.

## 설득 포인트
- 출근 전이나 점심 후처럼 직장인 루틴 안에서 복용 장면이 보여야 한다.
- 지금 챙기는 기본 비타민과 비교 포인트를 정리해 주면 스스로 판단하기 쉽다.
- 당장 결정하지 않아도 된다는 여유를 주는 말이 오히려 긍정적이다.
- 제품을 밀기보다 어떤 상황에 맞는 선택인지 설명해 주는 방식이 먹힌다.
- 정보는 짧고 정리되어 있어야 하며, 판단은 본인 몫으로 남겨야 한다.

## 절대 안 먹히는 것
- 마감 압박이나 즉시 결정을 요구하는 태도.
- 불안을 자극해 구매를 유도하는 방식.
- 직장인의 피로를 빌미로 과하게 몰아붙이는 화법.
- 비교 정보 없이 한 제품만 정답처럼 밀어붙이는 설명.
- 예산 고민을 가볍게 보는 태도.

## 예상 반론
- "왜 이렇게 빨리 결정하라고 하세요?" → 톤을 낮추고 선택권을 돌려줘야 한다.
- "일단 제가 더 비교해볼게요." → 그 태도를 존중해야 신뢰가 유지된다.
- "좋아 보여도 지금 바로 살 생각은 없어요." → 당장의 행동보다 판단 기준을 남겨야 한다.
- "현재 먹는 비타민이랑 뭐가 다른지 정도만 알고 싶어요." → 차이를 짧고 명확하게 말해줘야 한다.

## 관심 신호
- 구매 시점보다 복용법과 비교 기준을 다시 묻는다.
- 제품 이름을 메모하거나 퇴근 후 다시 볼 수 있게 정리해 달라고 한다.
- "부담 없이 비교해볼 수는 있겠네요"라는 말이 나오면 진지하게 보는 중이다.

## 이탈 신호
- 대답이 짧아지고 예민해지면 압박을 느낀 상태다.
- 혜택 마감 이야기에 표정이 굳으면 이미 마음이 닫히고 있다.
- 설명보다 영업 의도가 먼저 느껴지는 순간 바로 거리를 둔다.
