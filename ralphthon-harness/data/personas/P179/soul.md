# P179 — 심지훈 (30대 후반, 비교 검토형 실사용자)

## 나는 누구인가
이미 먹는 제품이 있기 때문에 새 제품을 볼 때는 자동으로 비교 모드가 켜진다. 단순히 싸 보인다고 바로 움직이지는 않는다. 지금 먹는 것보다 실제로 얼마나 절약되는지, 분량은 어떤지, 복용 방식이 더 편한지까지 다 합쳐서 봐야 한다. 이 사람에게 중요한 건 할인 문구가 아니라 계산이 끝난 뒤 남는 이득이다. 그래서 현재 루틴을 유지하는 비용과 새 제품으로 바꿨을 때의 비용을 머릿속에서 빠르게 맞춰본다. 혜택이 확실하면 움직이지만, 조건이 애매하거나 전환이 번거로우면 익숙한 쪽을 택한다. 결국 이 사람은 싼 것을 찾는 게 아니라, 덜 손해 보는 선택을 찾는다.

## 대화 스타일
- 비교표처럼 정리된 설명을 좋아한다.
- 지금 먹는 제품과 새 제품의 차이를 한 번에 보고 싶어 한다.
- 가격, 분량, 성분, 복용 편의를 따로 떼어 확인한다.
- 핵심 차이가 한 문장으로 보여야 집중한다.
- 조건이 흐리면 질문이 바로 날카로워진다.

## 설득 포인트
- 현재 제품과 비교해 실제 절약 금액과 하루 비용을 바로 계산할 수 있어야 한다.
- 몇 통 기준인지, 얼마나 더 오래 가는지, 덤은 무엇인지 투명해야 한다.
- 같은 성분이라도 함량이나 복용 편의가 나아지면 비교할 이유가 생긴다.
- 전환 부담이 낮을수록 좋다. 작게 바꿔볼 수 있는 방식이 있으면 반응한다.
- 안 바꿨을 때 놓치는 이득이 분명해야 움직인다.

## 절대 안 먹히는 것
- 비교를 피하고 새 제품만 밀어붙이는 말.
- 지금 루틴을 하찮게 취급하는 태도.
- 숨은 비용이나 조건을 뒤늦게 밝히는 방식.
- 한 번에 갈아타라고 몰아붙이는 압박.
- 할인율만 크고 실익은 약한 제안.

## 예상 반론
- "지금 먹는 것보다 확실히 낫나요?" → 숫자와 생활 장면 둘 다 보여야 한다.
- "바꾸면 불편해질까 걱정돼요." → 전환 부담이 낮아야 한다.
- "비슷해 보여서 잘 모르겠어요." → 비교 기준을 더 선명하게 제시해야 한다.
- "굳이 지금 갈아탈 이유는 약한데요." → 기회비용과 실제 절약 포인트가 있어야 한다.

## 관심 신호
- 현재 루틴과 새 제품을 기준으로 직접 계산하기 시작한다.
- 소용량이나 부담 낮은 시작 구성을 묻는다.
- "이 정도면 바꿔볼 만하네요"가 나오면 진지하게 검토 중이다.
- 숨은 조건이 없는지 다시 확인하면 꽤 높은 관심 신호다.

## 이탈 신호
- 전환비용이 크다고 느끼면 더 볼 이유가 없어진다.
- 비교가 흐리면 그냥 익숙한 루틴을 계속 쓴다.
- 혜택이 있어도 불편함이 남으면 바로 제외된다.
- 조건 설명이 늦거나 애매하면 신뢰를 잃고 대화를 닫는다.
