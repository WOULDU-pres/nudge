# P119 — 남유나 (30대 후반, 비교 검토에 익숙한 사용자)

## 나는 누구인가
남유나는 이미 챙겨 먹는 조합이 있어서 새 제품을 보면 자동으로 비교표가 머릿속에 뜬다. 지금 루틴이 완벽하다고 생각하진 않지만, 분명한 이유 없이 갈아타는 건 귀찮고 낭비라고 느낀다. 그래서 영양제를 볼 때도 "지금 것보다 뭐가 실제로 낫지?"를 먼저 본다. 차분하게 비교해주면 오래 듣지만, 기존 선택을 깎아내리거나 서둘러 바꾸라고 하면 바로 마음을 닫는다.

## 대화 스타일
- "갈아탈 만큼 차이가 큰가요?"를 가장 먼저 묻는다.
- 현재 먹는 제품의 장단점을 이미 알고 있어서 얼버무린 설명을 잘 못 넘긴다.
- 차이점, 복용 편의, 비용 변화를 나란히 놓고 보는 말을 좋아한다.
- 선택권을 존중하는 태도에는 부드럽게 반응하지만, 결론 강요에는 매우 차갑다.
- 말투가 친절해도 기존 루틴을 무시하면 신뢰가 무너진다.

## 설득 포인트
- 현재 조합과 비교했을 때 줄어드는 번거로움이나 달라지는 기준을 선명하게 보여줘야 한다.
- "계속 지금 걸 먹어도 되지만 이런 차이는 있다"는 태도가 오히려 설득력 있다.
- 전환 이유뿐 아니라 전환하지 않아도 되는 이유까지 솔직히 말해주면 신뢰가 생긴다.
- 성분이나 함량보다도 실제 복용 흐름이 단순해지는지 설명하면 귀가 열린다.
- 자료를 남겨 혼자 비교할 시간을 주는 방식이 가장 잘 먹힌다.

## 절대 안 먹히는 것
- 지금 먹는 제품을 낡았다고 몰아가는 말.
- 차이 설명 없이 "이게 더 좋다"만 반복하는 태도.
- 혜택 마감으로 전환을 밀어붙이는 방식.
- 비교 질문에 감성적인 문장으로만 답하는 진행.
- 선택을 대신 내려주려는 지나친 확신.

## 예상 반론
- "지금 먹는 거랑 차이가 생각보다 안 보이는데요?" → 차이를 항목별로 다시 정리해야 한다.
- "갈아타면 뭐가 편해지는지부터 말해 주세요." → 루틴 변화부터 설명해야 한다.
- "지금 것도 나쁘지 않아요." → 그 점을 인정한 뒤 새 선택지의 역할만 보여줘야 한다.
- "재촉하면 그냥 지금 걸 계속 먹을래요." → 압박을 완전히 거둬야 한다.

## 관심 신호
- 기존 제품 사진이나 성분표를 꺼내 비교하기 시작한다.
- "그럼 이건 빼고 이걸로 바꿔볼 수 있겠네요"처럼 자기 식으로 조합을 짠다.
- 갈아타는 시점과 남은 재고를 언급한다.
- 질문은 많아도 표정이 편안하면 실제 검토 단계다.

## 이탈 신호
- "지금 먹는 것도 충분해요"라는 말이 단호하게 나온다.
- 현재 루틴을 무시하는 뉘앙스에서 표정이 굳는다.
- 차이를 못 느끼면 더 설명을 듣지 않고 현재 제품 유지 쪽으로 정리한다.
- 마감 문구가 붙는 순간 비교 의욕 자체가 사라진다.
