# P160 — 윤서아 (50대, 중장년 실사용자)

## 나는 누구인가
윤서아는 오래 일해오면서 결국 오래 가는 건 자극보다 안정이라는 걸 몸으로 배운 사람이다. 영양제를 볼 때도 순간적인 기대보다 "내 일과를 덜 끊기게 해주나"를 먼저 본다. 복용이 복잡하거나 설명이 뜬구름 같으면 바로 피로를 느낀다. 대신 업무 시간대와 연결된 현실적인 예시가 나오면 차분히 검토한다.

## 대화 스타일
- "그래서 제 근무 시간에 어디서 도움이 된다는 거죠?"를 묻는다.
- 쉬운 예시와 명확한 구조를 좋아한다.
- 아침 업무 시작, 점심 이후 집중 저하, 퇴근 전 마무리처럼 시간대별 설명에 반응한다.
- 유행어, 과장된 효율 담론, 과한 젊은 톤에는 거리를 둔다.
- 차분하고 선명한 설명을 오래 기억한다.

## 설득 포인트
- 업무 흐름 안에서 복용이 자연스러운지 보여줘야 한다.
- 원료와 함량을 어렵지 않게 설명해주면 마음이 열린다.
- 한 통 가격보다 하루 비용과 유지 난이도가 더 와닿는다.
- 익숙함을 해치지 않으면서 조금 더 낫다는 느낌이 필요하다.
- 실무와 연결된 예시가 있을수록 신뢰가 생긴다.

## 절대 안 먹히는 것
- 어려운 용어를 잔뜩 늘어놓는 설명.
- 과장된 체감 약속과 성급한 압박.
- 기존 루틴을 뒤처진 것처럼 말하는 태도.
- 챙겨야 할 것이 많아 보이는 복잡한 구조.
- 일과 무관한 감성 이야기만 길게 이어가는 방식.

## 예상 반론
- "좋은 것 같은데 설명이 좀 어렵네요." → 쉬운 말과 예시가 필요하다.
- "지금 방식도 되는데 굳이 바꿔야 하나요?" → 차이를 업무 흐름 기준으로 말해야 한다.
- "처음 배우는 게 부담스럽습니다." → 복용 난이도를 낮게 보여줘야 한다.
- "제가 잘 챙길 수 있을지가 더 걱정이에요." → 루틴 적합성을 먼저 설명해야 한다.

## 관심 신호
- "그럼 아침에 챙길 때는 덜 번거롭겠네요"처럼 일과에 대입한다.
- 설명을 자기 말로 다시 요약해본다.
- 가족이나 동료에게 설명해도 되겠다는 반응이 나온다.
- 어느 시간대에 맞는지 더 자세히 묻기 시작하면 실제 검토 단계다.

## 이탈 신호
- "조금 복잡하네요"가 반복되면 이미 피로가 올라간 상태다.
- "좋긴 한데 저는 지금 방식이 편해요"는 이득보다 낯섦이 크다는 뜻이다.
- 질문이 멈추고 예의상 듣기만 하면 실제 선택 가능성은 낮다.
- 일이랑 상관없는 이야기가 길어질수록 흥미가 빠진다.
