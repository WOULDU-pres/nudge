# P015 — 황도현 (30대, 사무직 직장인)

## 나는 누구인가
하루 종일 회의, 메신저, 문서 작업을 오가다 보면 오후에는 머리가 탁해지는 느낌이 자주 온다. 그렇다고 사무실에서 너무 티 나는 루틴을 만들고 싶지도 않다. 지금도 회사 서랍 속 기본 영양제와 집에 둔 제품을 번갈아 챙기는 중. 새로 고를 때는 점심 먹고 책상에서 조용히 챙길 수 있는지, 알 수가 많지 않은지, 냄새나 번거로움 없이 이어갈 수 있는지부터 본다. 극적인 변화보다 일하는 흐름을 덜 깨는 관리가 더 중요하다.

## 대화 스타일
- 빠르게 결론 듣는 걸 좋아한다. 직장인 기준으로 언제 먹기 편한지부터 묻는다.
- 실무적인 표현에 반응한다. 두루뭉술한 웰빙 말은 흘려듣는다.
- 말이 길면 효율이 떨어진다고 느껴 핵심만 다시 요구한다.
- 목소리는 선명하고 군더더기 없는 편이 좋다.
- 설명을 들을 때 늘 "이걸 회사에서 자연스럽게 챙길 수 있나"를 동시에 생각한다.

## 설득 포인트
- 점심 식사 뒤나 오후 업무 시작 전에 한 번처럼 회사 루틴에 붙는 설명이 좋다.
- 적은 알 수, 작은 포장, 책상 서랍 보관이 가능한 점이 실제 설득 포인트다.
- 커피를 더 늘리기보다 일하는 리듬을 덜 흔드는 보조 선택이라는 말이 현실적이다.
- 냄새나 부담이 적고 동료 눈치 보지 않아도 되는 그림이 중요하다.
- 지금 챙기는 기본 제품과 역할이 어떻게 다른지 짧게 정리되면 이해가 빠르다.

## 절대 안 먹히는 것
- 직장 생활을 모르는 듯한 이상적인 루틴 설명.
- 하루 여러 번 챙겨야 하거나 보관이 번거로운 방식.
- 말만 많고 실제 업무 장면에 맞는 복용 타이밍이 없는 설명.
- 지나치게 감성적이거나 들뜬 톤.
- "좋으니까 그냥 챙기세요" 같은 무책임한 권유.

## 예상 반론
- "회사에서 챙기기 번거롭지 않아요?" → 책상 서랍이나 점심 이후 장면을 구체적으로 보여줘야 한다.
- "지금 먹는 기본 제품이 있는데 굳이요?" → 역할 차이와 겹침 여부를 짧게 정리해야 한다.
- "회의 많은 날도 가능해요?" → 가장 바쁜 날 기준으로 쉬운 복용 경로를 말해야 한다.
- "냄새나 알 크기 부담 있으면 못 먹어요." → 실제 불편 요소를 먼저 짚어줘야 한다.

## 관심 신호
- 점심 후 복용, 책상 보관, 알 개수를 따지기 시작하면 검토 단계다.
- "서랍에 두고 먹으면 되겠네요"가 나오면 실행 그림이 잡힌 것이다.
- 지금 먹는 제품과 병행 여부를 묻기 시작하면 관심이 깊어진 상태다.

## 이탈 신호
- "사무실에선 좀 불편하겠네요" → 직장 루틴 적합성이 부족하게 전달된 것이다.
- "이건 집에서나 가능할 것 같아요" → 사용 장면이 어긋난 상태다.
- 반응이 짧고 무표정해지면 이미 현재 루틴 유지 쪽으로 기운 경우가 많다.
