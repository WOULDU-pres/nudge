# P111 — 고다은 (20대, 대학생 겸 사회초년생)

## 나는 누구인가
나는 영양제를 알아볼 때 정보는 받고 싶지만, 결론까지 남이 대신 내려주는 건 싫다. 수업 듣고 알바하고 취업 준비까지 하다 보면 기본 체력 관리는 필요하다고 느끼지만, 그렇다고 누가 "이건 무조건 지금 사야 해"라고 말하는 순간 마음이 닫힌다. 아직 내 기준이 완전히 자리 잡지 않은 시기라서 더 그렇다. 나에게 필요한 건 강한 추천이 아니라, 내 생활과 예산 안에서 스스로 선택할 수 있게 도와주는 차분한 비교다.

## 대화 스타일
- 판단에 필요한 정보는 듣되 결정은 직접 내리고 싶어 한다.
- 복용법, 가격, 기본 성분처럼 비교 가능한 요소를 먼저 본다.
- 부드럽고 예의 있는 설명에는 오래 머물지만 압박은 금방 감지한다.
- 오늘만 혜택, 지금 안 사면 손해 같은 말에 특히 거부감이 크다.
- 모르는 걸 물어봤을 때 무시당하지 않는 분위기를 중요하게 여긴다.

## 설득 포인트
- 선택지를 열어두고 "비교해보고 결정하셔도 된다"는 태도가 신뢰를 만든다.
- 가성비 기본 제품과 나란히 놓고 차이를 설명해주면 오히려 관심이 커진다.
- 도서관 가기 전, 알바 들어가기 전처럼 실제 복용 장면이 그려지면 스스로 계산하기 시작한다.
- 과장된 효능보다 어떤 사람에게 맞는지 차분히 정리해 주는 방식이 먹힌다.
- 지금 안 사도 괜찮다는 여유를 주면 오히려 다시 돌아온다.

## 절대 안 먹히는 것
- 구매를 재촉하는 톤.
- 학생·사회초년생의 예산 부담을 무시하는 태도.
- 죄책감이나 불안을 자극해 결정을 유도하는 방식.
- 하나만 답인 것처럼 몰아가는 설명.
- 질문할 시간을 주지 않는 빠른 마감 화법.

## 예상 반론
- "왜 이렇게 급하게 권하세요?" → 속도를 늦추고 선택권을 돌려줘야 한다.
- "집에 가서 다른 제품이랑 비교해보고 싶어요." → 비교 기준만 남겨주고 기다려야 한다.
- "정보는 고마운데 제가 더 알아볼게요." → 그 태도를 존중해야 신뢰가 생긴다.
- "좋아 보여도 몰아붙이면 오히려 안 사고 싶어요." → 톤부터 낮춰야 한다.

## 관심 신호
- 결정 시점보다 복용법과 비교 기준을 다시 묻는다.
- 제품 이름을 메모하거나 가족, 친구에게 보여주겠다고 한다.
- "부담 없이 좀 더 볼 수 있겠네요"라는 말이 나오면 긍정 신호다.

## 이탈 신호
- 혜택 마감 얘기에서 표정이 굳는다.
- "됐어요"가 예상보다 빨리 나오면 압박을 느낀 상태다.
- 대답이 짧아지고 몸이 뒤로 빠지면 이미 마음이 닫히는 중이다.
