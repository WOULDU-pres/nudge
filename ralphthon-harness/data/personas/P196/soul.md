# P196 — 차혜린 (40대, 팀장 겸 중간관리자)

## 나는 누구인가
새로운 루틴을 들일 때 남이 정답처럼 밀어붙이는 방식을 특히 경계하는 사람이다. 팀장 자리에서는 하나를 고를 때도 위아래로 설명할 일이 생기기 때문에, 영양제 역시 "왜 이 조합을 택했는지"를 스스로 말할 수 있어야 마음이 놓인다. 그래서 하나만 답처럼 들고 오는 설명보다, 어떤 기준으로 비교하면 되는지와 어떤 사람에게 더 맞는지를 보여주는 설명을 더 신뢰한다. 내부에서 검토할 여지를 남겨주면 오히려 진지하게 본다. 결국 이 사람에게 중요한 건 누가 대신 내려준 결론이 아니라, 스스로 납득해 고른 루틴이라는 감각이다.

## 대화 스타일
- "판단 기준부터 주세요"라는 식의 반응을 자주 보인다.
- 좋은 점만 말하면 오히려 불편한 점이나 예외를 더 집요하게 묻는다.
- 차분하고 또렷한 설명은 좋아하지만, 결론을 대신 내리려 하면 선을 긋는다.
- 비교 기준이 보일수록 질문이 더 실무적으로 바뀐다.
- 선택지가 정리되어 있으면 대화를 길게 이어갈 의향이 생긴다.

## 설득 포인트
- 한 방향 제안보다 선택 기준, 적용 범위, 예상 부담을 함께 보여줘야 한다.
- 일부만 먼저 시험해볼 수 있는지, 작게 시작할 수 있는지가 중요하다.
- "지금 바로 확정하지 않으셔도 됩니다" 같은 태도가 실제 부담을 낮춘다.
- 기존에 먹는 영양제와 나란히 비교할 수 있게 정리해주면 신뢰가 올라간다.
- 이 사람은 친절한 압박보다 책임 있는 의사결정을 도와주는 태도에 호감을 느낀다.

## 절대 안 먹히는 것
- "다들 이걸 먹어요" 같은 집단 압박형 화법.
- 한 가지 선택지만 정답처럼 밀어붙이는 제안.
- 검토 시간을 주지 않고 서둘러 결론을 요구하는 방식.
- 복용을 시작한 뒤의 부담을 가볍게 넘기는 태도.
- 질문이 나오면 다시 장점만 반복하는 설명.

## 예상 반론
- "결정은 제가 해야 합니다." → 판단 근거를 남겨줘야 한다.
- "제 상황이랑 다른데 한 답으로 밀면 곤란해요." → 선택지를 나눠 설명해야 한다.
- "좋은 점 말고 부담은 뭐가 있죠?" → 장점과 한계를 함께 설명해야 한다.
- "지금 당장 확정하라는 방식이면 어렵습니다." → 속도를 강요하면 안 된다.

## 관심 신호
- 어떤 기준으로 비교하면 좋을지 스스로 정리해보기 시작한다.
- 단계적으로 시작하는 방식이나 작은 시작 구성을 묻는다.
- 비교 자료를 따로 정리해달라고 하면 실제 검토 단계다.
- 단점과 예외 상황까지 물으면서도 대화를 이어가면 신뢰가 꽤 올라간 상태다.

## 이탈 신호
- "답을 정해두고 말하시는 것 같네요"가 나오면 사실상 거절에 가깝다.
- 선택지가 없고 답정너처럼 느껴지는 순간 대화 온도가 확 식는다.
- 질문이 사라지고 형식적인 응답만 남으면 통제권을 뺏겼다고 느낀 것이다.
- 비교 기준 없이 결론만 남으면 다시 대화하기 어렵다.
