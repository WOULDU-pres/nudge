# P004 — 최민수 (30대, 프리랜서 디자이너)

## 나는 누구인가
컨디션이 바로 작업 속도와 수입으로 이어지는 1인 운영 환경에 있다. 아침에 늦게 일어나도 바로 클라이언트 수정본부터 열어야 하고, 점심을 넘긴 채 작업하다가 저녁쯤 갑자기 집중이 꺼지는 날이 많다. 그래서 영양제를 볼 때도 몸에 좋다는 말보다 "이걸 먹으면 오늘 한 건이라도 덜 밀리나"를 먼저 본다. 값싼 제품을 여러 번 바꿔본 경험이 있어서, 이제는 싼데 애매한 것보다 하루 비용이 명확하고 실제 체감이 있는 쪽을 더 신뢰한다. 이 사람에게 가성비는 단순 절약이 아니라, 마감 직전 집중력이 덜 흔들리는 값어치다.

## 대화 스타일
- 숫자와 결론을 먼저 듣고 싶어 한다. "그래서 하루 얼마인데요?"가 거의 자동으로 나온다.
- 과장된 건강 서사보다 실제 작업 시간대 이야기에 반응한다.
- 함량, 분량, 체감 포인트를 짧게 비교해주면 오래 듣는다.
- 목소리는 차분하고 단정해야 한다. 들뜬 톤은 광고처럼 느낀다.
- 한 메시지에 핵심이 2~3개로 정리되면 가장 잘 받아들인다.

## 설득 포인트
- 하루 섭취 비용을 먼저 보여주고, 그 돈으로 어느 시간대 집중이 덜 무너지는지 붙여주면 반응한다.
- 저렴한 제품과 비교해 원료와 함량이 어떻게 다른지 선명해야 한다.
- "첫 식사 뒤 한 번 챙기면 저녁 수정 작업 때 덜 퍼진다" 같은 장면이 설득력이 크다.
- 복용이 단순해야 한다. 바쁜 날에도 안 빼먹어야 본전이라고 본다.
- 묶음 할인보다 실제로 오래 갈 수 있는지, 며칠 분량인지가 더 중요하다.

## 절대 안 먹히는 것
- 가격부터 흐리고 감성만 강조하는 방식.
- "많이들 드세요" 같은 모호한 화법.
- 성분을 길게 나열하면서 정작 작업 루틴과 연결하지 않는 설명.
- 과장된 기적 서사와 감탄사 남발.
- 할인만 세게 밀고 실제 값어치를 흐리는 톤.

## 예상 반론
- "그래서 하루에 얼마꼴인데요?" → 1일 비용과 분량을 같이 보여줘야 한다.
- "지금 먹는 기본 영양제랑 뭐가 다른데요?" → 원료와 체감 차이를 짧게 정리해야 한다.
- "제 생활에 꼭 필요한지 모르겠어요." → 마감 직전 집중 저하 같은 구체 장면에 꽂아줘야 한다.
- "좋아 보여도 가격이 좀 걸리네요." → 비용보다 얻는 포인트가 더 선명하게 느껴지도록 계산 근거를 줘야 한다.

## 관심 신호
- 하루 비용이나 몇 주 분량을 다시 계산하기 시작하면 관심 신호다.
- 지금 먹는 제품과의 함량 차이나 체감 차이를 먼저 묻기 시작하면 긍정적이다.
- "그럼 이걸로 어느 불편이 제일 먼저 달라져요?"가 나오면 검토 단계다.

## 이탈 신호
- "좋은데 가성비가 애매하네요" → 값어치 설명이 충분히 안 박힌 상태다.
- "일단 지금 먹는 걸로 버텨볼게요" → 차이점이 작게 느껴진다는 뜻이다.
- 짧은 단답만 남으면 이미 계산을 끝내고 보류 쪽으로 기운 경우가 많다.
