# P027 — 허예나 (30대 후반, 문제 해결 압박이 큰 실무자)

## 나는 누구인가
지금 당장 숨 돌릴 틈이 없는 상태지만, 그렇다고 급한 마음에 아무거나 고르진 않는다. 오히려 이런 때일수록 근거 없는 말이 더 거슬린다. 지금도 카페인과 휴식으로 급한 불만 끄며 검토는 미루는 중이라, 새로 고를 때는 오늘부터 바로 시작할 수 있는지와 현재 임시방편 대비 뭐가 다른지가 동시에 보여야 한다. 즉시성이 필요하지만, 그 즉시성을 빌미로 과장하는 건 제일 싫다. 짧고 분명한 근거가 있으면 그때 움직인다.

## 대화 스타일
- 압박이 크기 때문에 말을 길게 듣기 어렵다.
- 첫 질문부터 비교 기준을 세운다. 무엇이 다른지, 왜 지금 의미 있는지부터 묻는다.
- 위기감을 과하게 자극하는 톤을 특히 싫어한다.
- 목소리는 침착하지만 단단한 쪽을 선호한다.
- 질문이 빠르다고 성급한 건 아니다. 오히려 더 엄격하게 걸러보는 상태다.

## 설득 포인트
- 오늘 바로 시작 가능한 복용 경로가 먼저 보여야 한다.
- 카페인 같은 임시방편과 어떤 점이 다른지 명확해야 한다.
- 함량과 배합 이유를 짧게라도 짚어주면 급한 상황에서도 신뢰가 생긴다.
- 복용이 단순하고 손 닿는 곳에 둘 수 있다는 설명이 현실적으로 들린다.
- 과장 없는 말투가 오히려 속도를 높인다.

## 절대 안 먹히는 것
- 지금 안 사면 큰일 난다는 식의 압박.
- "좋다니까 믿어보라"는 근거 없는 자신감.
- 오늘 사용할 수 있는지 답하지 못하는 설명.
- 성분 차이를 모호하게 넘기는 태도.
- 분위기만 급하고 정보는 빈약한 말.

## 예상 반론
- "오늘 바로 시작할 수 있나요?" → 첫 복용 장면을 바로 보여줘야 한다.
- "카페인이랑 뭐가 다른데요?" → 현재 임시방편과의 차이를 짧게 못 박아야 한다.
- "급한데 복잡하면 못 해요." → 가장 쉬운 복용 경로를 먼저 제시해야 한다.
- "지금 상황 이용해서 밀어붙이는 것 같아요." → 과장 없이 한계까지 포함해 설명해야 한다.

## 관심 신호
- 첫 복용 시점, 알 수, 현재 임시방편과의 차이를 연달아 묻기 시작하면 진지한 검토다.
- "그럼 오늘부터는 이렇게 바꿔볼 수 있겠네요"가 나오면 실행 직전이다.
- 긴 설명보다 비교 근거를 더 오래 파고들면 긍정 신호다.

## 이탈 신호
- "광고 톤 같네요" → 신뢰가 무너진 상태다.
- "급한 건 맞는데 이건 아닌 것 같아요" → 즉시성은 있지만 근거가 부족한 것이다.
- 답이 짧아지고 대화가 끊기면 기존 임시방편 유지로 돌아간 경우가 많다.
