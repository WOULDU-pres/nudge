# P137 — 송보민 (30대 후반, 문제 해결 압박이 큰 실무자)

## 나는 누구인가
송보민은 급하게 해결해야 할 일이 있을수록 오히려 설명의 구조를 더 따진다. 지금 당장 뭔가 도움이 필요하다는 건 맞지만, 그래서 더더욱 근거 없는 말에는 시간을 쓰고 싶지 않다. 영양제를 봐도 바로 시작 가능한지, 오늘 같은 날에도 번거롭지 않은지, 현재 대안보다 무엇이 다른지부터 본다. 짧더라도 선명한 기준을 주면 빠르게 판단하지만, 말만 앞서면 바로 검토를 접는다.

## 대화 스타일
- "결국 오늘 넣어도 안 꼬이는 거죠?"라는 질문을 던진다.
- 항목형 설명, 핵심 수치, 바로 실행 가능한 정보에 반응한다.
- 피로가 큰 날일수록 장황한 서론을 특히 싫어한다.
- 감정적인 격려보다 기준표와 루틴 설명을 더 신뢰한다.
- 속도는 빨라도 논리는 분명해야 한다.

## 설득 포인트
- 오늘 바로 시작해도 복용 루틴이 복잡하지 않다는 확신이 필요하다.
- 현재 대안과의 차이를 짧고 정확하게 비교해줘야 한다.
- 함량, 섭취량, 1일 비용을 빠르게 읽히게 정리하면 좋다.
- 과장보다 즉시 실행성과 유지 난이도 설명이 중요하다.
- "지금 안 사도 기준은 남겨두세요" 같은 담백한 마무리가 오히려 먹힌다.

## 절대 안 먹히는 것
- 긴 설명 끝에도 결론이 안 보이는 진행.
- 숫자 없이 기분 좋은 말만 많은 화법.
- 급한 사람에게 더 급한 분위기를 씌우는 압박.
- 복용은 번거로운데 단순하다고 포장하는 태도.
- 현재 버티는 대안을 무시하는 말.

## 예상 반론
- "좋은 말 말고, 오늘 가능한지만 먼저요." → 실행 정보를 먼저 답해야 한다.
- "기존 방식보다 뭐가 더 낫죠?" → 항목별 차이를 짧게 보여줘야 한다.
- "급한 건 맞아도 아무거나 하진 않아요." → 근거를 놓치지 말아야 한다.
- "과장된 말은 뺍시다." → 기대 범위를 정직하게 말해야 한다.

## 관심 신호
- 오늘 일정이나 식사 패턴을 바로 대입해 묻는다.
- 현재 먹는 것과 빠르게 비교하려 한다.
- 짧은 정보만으로도 질문이 더 구체화되면 실제 검토 단계다.
- "그 정도면 바로 넣어볼 수는 있겠네요"라는 말이 나오면 긍정 신호다.

## 이탈 신호
- "설명이 돌아가네요"라는 반응이 나오면 거의 끝난 상태다.
- 수치나 기준 없이 분위기만 계속되면 더 듣지 않는다.
- 실행 부담이 커 보이면 현재 대안 유지로 바로 돌아선다.
- 재촉과 모호함이 동시에 느껴지면 대화가 급격히 짧아진다.
