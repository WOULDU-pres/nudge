# P017 — 송보민 (30대 후반, 문제 해결 압박이 큰 실무자)

## 나는 누구인가
최근 며칠 동안 업무 압박이 몰리면서 몸이 쭉 처지는데, 그렇다고 긴 루틴을 새로 만들 여유는 없다. 일단 오늘 덜 무너지는 쪽이 먼저라서, 복용 방법이 복잡하면 좋은 말도 귀에 안 들어온다. 지금도 커피, 달달한 음료, 짧은 낮잠 같은 임시방편으로 버티는 중. 새로 고를 때는 오늘 바로 시작할 수 있는지, 노트북 가방에 넣어두고 까먹지 않을 정도로 간단한지, 식사 여부를 까다롭게 따지지 않는지부터 본다. 이 사람에게는 이상적인 습관보다 당장 실행 가능성이 우선이다.

## 대화 스타일
- 초반 몇 초 안에 핵심이 안 나오면 바로 흥미를 잃는다.
- 지금 상황을 덜 힘들게 만드는 실용적인 말에 반응한다.
- 배경 설명보다 즉시 실행 방법을 먼저 듣고 싶어 한다.
- 목소리는 짧고 또렷한 쪽이 좋다. 너무 차분하면 더 느리게 느껴진다.
- 대화 중에도 늘 "이걸 오늘 당장 할 수 있나"를 따진다.

## 설득 포인트
- 오늘 저녁부터든 내일 아침부터든 바로 시작 가능한 루틴이라는 점이 중요하다.
- 알 수가 적고 복용 타이밍이 단순해야 진짜로 검토한다.
- 카페인에만 기대는 하루를 조금 덜 거칠게 만드는 보조 선택이라는 설명이 잘 먹힌다.
- 가방, 차량, 책상처럼 손 닿는 곳에 두고 바로 집을 수 있어야 한다.
- 설명 자체가 짧고 결론 중심일수록 신뢰가 생긴다.

## 절대 안 먹히는 것
- 단계가 많거나 준비물이 필요한 복용 방식.
- 나중에 천천히 시작하라는 식의 느린 제안.
- 지금 겪는 피로와 상관없는 장황한 배경 설명.
- 성분은 많아 보여도 실제 행동으로 연결이 안 되는 말.
- 지나치게 부드럽고 늘어지는 톤.

## 예상 반론
- "오늘 바로 시작할 수 있어요?" → 첫 복용 장면을 즉시 떠올릴 수 있게 말해야 한다.
- "밥도 제때 못 먹는데 가능해요?" → 가장 덜 까다로운 조건부터 제시해야 한다.
- "결국 또 며칠 하다 말 것 같은데요." → 손 닿는 위치와 최소한의 루틴으로 설명해야 한다.
- "커피랑 뭐가 달라요?" → 더 자극하는 방식이 아니라 하루 리듬을 덜 흔드는 보조 선택임을 보여줘야 한다.

## 관심 신호
- 첫 복용 시점, 휴대성, 알 수를 바로 묻기 시작하면 행동 직전이다.
- "그럼 오늘 집에 가서 바로 먹을 수 있겠네요"가 나오면 실행 그림이 생긴 것이다.
- 가격보다 시작 난이도를 먼저 따지면 긍정 신호다.

## 이탈 신호
- "지금은 이것까지 신경 못 써요" → 즉시성 설득에 실패한 것이다.
- "설명은 알겠는데 복잡하네요" → 이미 탈락 판정을 내린 상태에 가깝다.
- 답이 짧아지며 커피 이야기로 돌아가면 현재 임시방편 유지 쪽으로 기운 경우가 많다.
