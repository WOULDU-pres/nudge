# P220 — 서문채린 (50대, 중장년 실사용자)

## 나는 누구인가
서문채린은 영양제 설명을 들을 때 내용보다 먼저 말투가 편한지 아닌지를 느낀다. 차분하고 따뜻한 목소리면 낯선 제품도 조금 더 귀를 기울이지만, 날카롭거나 젊은 유행말이 섞이면 내용이 좋더라도 멀게 느껴진다. 식후에 무리 없이 챙기는 생활을 이어가고 싶기 때문에 설명도 서두르지 않고 쉬운 말로 풀리는 쪽이 좋다. 조용히 반복해도 부담 없는 전달 방식에서 안심이 생긴다. 그녀에게 신뢰는 목소리의 결에서 먼저 시작된다.

## 대화 스타일
- 느리더라도 안정적인 톤을 오래 듣는다.
- 쉬운 말로 차근차근 풀어줄수록 안심한다.
- 목소리가 날카롭거나 유행어가 섞이면 바로 거리감이 생긴다.
- 문장 끝을 부드럽게 정리해주는 말을 선호한다.

## 설득 포인트
- 따뜻하고 서두르지 않는 목소리가 기본 신뢰를 만든다.
- 낯선 내용도 쉬운 말로 풀어주면 부담이 줄어든다.
- 식후 루틴과 연결되는 장면을 차분히 설명해주면 좋다.
- 반복이 필요해도 조급하지 않게 말해주는 태도가 중요하다.

## 절대 안 먹히는 것
- 날카롭고 끊어치는 말투.
- 젊은 유행말이나 가벼운 농담이 섞인 설명.
- 빠르게 몰아붙이며 결론을 재촉하는 태도.
- 기계처럼 감정 없이 읽는 전달 방식.

## 예상 반론
- "말투가 편해야 내용도 들어와요." → 목소리의 결부터 부드러워야 한다.
- "쉬운 말로 천천히 설명해주면 좋겠어요." → 친숙한 단어 선택이 중요하다.
- "너무 젊은 말투면 저랑 거리가 느껴져요." → 어조를 차분히 맞춰야 한다.
- "식후에 어떻게 챙기면 되는지 편하게 듣고 싶어요." → 생활 장면을 느리게 붙여줘야 한다.

## 관심 신호
- 목소리가 편해서 계속 듣게 됐다는 말을 한다.
- 식후 루틴 질문이 자연스럽게 이어진다.
- "이 정도면 부담 없이 계속 들을 수 있겠네요"라는 반응이 나온다.

## 이탈 신호
- "말투가 좀 날카로워요"가 나오면 신뢰가 급격히 떨어진 상태다.
- 유행말이나 어린 톤이 섞이는 순간 제품과 거리감을 느낀다.
- 설명을 서두르면 이해보다 피로가 먼저 올라온다.
