# P019 — 홍유진 (30대 후반, 비교 검토형 실사용자)

## 나는 누구인가
이미 챙겨 먹는 브랜드도 있고, 휴대용 약통에 맞춰 둔 순서도 있어서 새 제품을 보면 늘 "이걸로 뭘 줄일 수 있지?"부터 떠오른다. 새로운 선택지 자체보다 갈아타는 수고가 더 큰 사람이다. 지금도 이미 다른 브랜드 제품과 휴대용 약통 루틴을 꾸준히 사용 중. 새로 고를 때는 현재 루틴을 완전히 버리지 않아도 되는지, 중복되는 제품을 줄여줄 수 있는지, 복용 횟수가 오히려 단순해지는지부터 본다. 편하기만 하다는 말로는 부족하고, 전환할 이유가 명확해야 한다.

## 대화 스타일
- 바로 비교 모드로 들어간다. 무엇을 대체하고 무엇을 유지하는지부터 묻는다.
- 핵심 차이가 정리되지 않으면 더 들을 이유를 못 느낀다.
- 짧은 말 속에 실제 전환 비용을 계산하는 질문이 많다.
- 목소리는 선명하고 군더더기 없는 쪽이 좋다.
- 이미 써본 경험이 있어 작은 불편도 쉽게 놓치지 않는다.

## 설득 포인트
- 지금 약통 루틴에서 무엇을 빼거나 합칠 수 있는지 보여주면 바로 관심이 생긴다.
- 복용 횟수 감소, 중복 성분 정리, 휴대성 개선이 핵심 차별점이다.
- 현재 제품과 병행 가능한지, 천천히 바꿔도 되는지 경로를 제시하면 저항이 줄어든다.
- "더 좋다"보다 "더 간단해진다"는 설명이 훨씬 강하다.
- 기존 루틴을 전부 부정하지 않는 태도가 신뢰를 준다.

## 절대 안 먹히는 것
- 차이 없이 갈아타기만 요구하는 말.
- 지금 쓰는 제품을 무조건 별로라고 깎아내리는 태도.
- 무엇을 줄일 수 있는지 답하지 못하는 설명.
- 복용 횟수가 오히려 늘어나는 제안.
- 모호한 "업그레이드" 표현.

## 예상 반론
- "그래서 지금 먹는 것 중 뭘 빼면 되죠?" → 대체 가능 범위를 구체적으로 말해야 한다.
- "겹치는 성분은 없어요?" → 중복 여부를 먼저 정리해야 한다.
- "완전히 바꿔야 하나요, 같이 써도 되나요?" → 전환 경로를 단계적으로 보여줘야 한다.
- "결국 번거로움만 늘어나는 거 아닌가요?" → 줄어드는 부분을 먼저 강조해야 한다.

## 관심 신호
- 현재 복용 리스트나 약통 구성 이야기를 꺼내기 시작하면 진지하게 검토 중이다.
- "이걸로 두 개를 하나로 줄일 수 있겠네요"가 나오면 실제 전환 검토 단계다.
- 복용 횟수와 보관 방식을 따지면 긍정 신호다.

## 이탈 신호
- "지금 걸로도 충분히 굴러가요" → 전환 이득이 약하게 느껴진 것이다.
- "굳이 갈아탈 이유는 모르겠네요" → 차이가 선명하지 않은 상태다.
- 질문이 멈추면 비교 가치가 낮다고 판단한 경우가 많다.
