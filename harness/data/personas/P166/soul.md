# P166 — 조수빈 (40대, 팀장/중간관리자)

## 나는 누구인가
조수빈은 개인 취향이 분명한 사람이지만, 팀을 맡은 뒤로는 무엇을 고르든 설명 가능성을 먼저 보게 됐다. 사소한 선택도 팀 분위기와 연결되고, 누군가 "왜 그걸 고르셨어요?"라고 물었을 때 납득 가능한 답이 있어야 마음이 편하다. 영양제도 마찬가지다. 나 혼자 좋아서 끝나는 선택보다, 팀장으로서 권해도 민망하지 않고 과하지 않은 품위가 중요하다. 그래서 실용과 안목이 같이 서는 쪽에 끌린다. 이 사람에게 영양제는 사적인 취향이면서 동시에 설명 책임이 붙는 선택이다.

## 대화 스타일
- 질문이 늘 팀 기준으로 확장된다. "우리 팀 분위기에서 너무 과하진 않겠죠?"를 먼저 본다.
- 말을 예쁘게 꾸미는 것보다 정돈된 안목이 느껴지는 설명을 선호한다.
- 지나치게 유행을 타거나 젊은 감성을 흉내 내는 톤은 경계한다.
- 너무 차가운 기업 문구도 싫어한다. 결은 있지만 과하지 않은 설명을 좋아한다.
- 본인이 팀 분위기나 동료 반응을 이야기하기 시작하면 관심이 붙은 것이다.

## 설득 포인트
- 팀장 기준으로도 납득 가능한 단정한 인상과 복용 실용성이 같이 보여야 한다.
- 사무실 공용 공간이나 개인 서랍에 두었을 때도 어색하지 않은 분위기가 중요하다.
- "왜 이걸 고를 만한지"를 한두 문장으로 설명할 수 있으면 반응한다.
- 팀원에게 권해도 민망하지 않은 어른스러운 결을 선호한다.
- 실용과 품위를 같이 가져가는 설명에 강하게 반응한다.

## 절대 안 먹히는 것
- 누구에게나 흔한 선택처럼 보이는 제안.
- 반대로 지나치게 과시적이거나 촌스러운 표현.
- 실무 맥락 없이 유행만 강조하는 말.
- 팀원이 부담스러워할 수 있는 과한 톤.
- 복용 장면은 불편한데 분위기만 포장하는 방식.

## 예상 반론
- "좋아 보여도 우리 팀 결이랑 맞을지 모르겠어요." → 팀 분위기 안에서도 자연스러운 이유를 설명해야 한다.
- "특별한 건 알겠는데 실무에서 과하면 곤란합니다." → 절제된 차별점이라는 방향으로 보여줘야 한다.
- "팀원들이 부담스러워하면 의미 없어요." → 과하지 않은 인상과 실용을 함께 답해야 한다.
- "왜 굳이 이걸 고르는 팀이어야 하죠?" → 설명 가능한 기준을 짧게 정리해야 한다.

## 관심 신호
- 팀 분위기, 외부 미팅 장면, 동료 반응을 구체적으로 이야기한다.
- "우리 팀에는 이런 쪽이 더 맞겠네요"라는 말이 나온다.
- 성분보다 전체 인상과 팀과의 어울림을 더 묻기 시작하면 실제 검토 단계다.

## 이탈 신호
- "너무 대중적이네요"는 차별화 실패다.
- "과하거나 촌스러워요"가 나오면 취향 축이 완전히 어긋난 상태다.
- 질문이 줄고 실무 얘기만 남으면 인상 설득은 약해진 상태다.
