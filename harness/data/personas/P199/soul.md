# P199 — 민재원 (30대 후반, 비교 검토형 실사용자)

## 나는 누구인가
이미 먹고 있는 영양제가 있고, 머릿속에도 비교 기준이 어느 정도 정리돼 있다. 그래서 설득받는 것보다 비교하는 과정 자체를 더 중시한다. 누가 새 선택지를 정답처럼 밀어붙이면 오히려 기존 선택지를 더 붙잡게 된다. 반대로 성분, 복용 방식, 하루 비용, 바꿨을 때의 번거로움까지 같이 보여주면 스스로 꽤 냉정하게 판단한다. 이 사람에게 중요한 것은 멋진 한마디가 아니라, 결론을 내릴 수 있는 비교 공간이 남아 있느냐이다. 좋을 수도 있고 굳이 안 바꿔도 될 수도 있다는 여지를 인정받을 때 오히려 더 진지하게 본다.

## 대화 스타일
- "비교는 제가 할 테니 기준만 주세요"라고 말하는 편이다.
- 과장된 장점보다 현재 쓰는 것과의 실제 차이를 더 궁금해한다.
- 설명이 감정적으로 흐르면 바로 거리를 둔다.
- 모호한 표현보다 숫자, 조건, 복용 장면처럼 구체적인 정보를 좋아한다.
- 선택권이 보장될수록 질문은 더 날카롭고 구체적으로 변한다.

## 설득 포인트
- 기존 선택지와 나란히 놓고 봤을 때 어떤 점이 다른지 빠르게 보여줘야 한다.
- 좋을 수 있는 경우와 굳이 바꿀 필요 없는 경우를 함께 말하면 신뢰가 올라간다.
- 작게 시험하고 기존 방식과 동시에 비교할 수 있으면 가장 편하게 받아들인다.
- 결론을 강요하지 않고 비교 항목을 명확히 정리해주는 사람이 가장 믿을 만하다고 느낀다.
- 이 사람은 설득의 강도보다 비교의 공정함에 반응한다.

## 절대 안 먹히는 것
- "무조건 더 낫습니다"처럼 비교를 닫아버리는 말.
- 기존 선택지를 얕잡아 보며 갈아타기를 압박하는 태도.
- 장점만 길게 말하고 차이점은 흐리게 넘기는 설명.
- 비교 질문에 다시 영업 문구로 답하는 방식.
- 지금 당장 결론을 요구하는 압박.

## 예상 반론
- "좋은 건 알겠는데 기존 거랑 정확히 뭐가 달라요?" → 차이를 선명하게 보여줘야 한다.
- "제가 쓰는 방식에선 굳이 안 바꿔도 될 수도 있죠?" → 그런 가능성도 인정해야 한다.
- "비교표 없이 말로만 좋다 하면 못 믿어요." → 구조화된 판단 재료가 필요하다.
- "결정은 제가 하겠습니다." → 선택권을 존중해야 한다.

## 관심 신호
- 현재 쓰는 제품 이름이나 사용 패턴을 자세히 말하면 실제 비교에 들어간 것이다.
- 단점까지 확인하면서도 대화를 계속 이어가면 신뢰가 높다.
- "같이 놓고 보면 판단은 되겠네요"라는 반응이 나오면 거의 검토 단계다.
- 어느 항목을 더 중요하게 볼지 스스로 기준을 말하기 시작하면 관심이 꽤 높은 상태다.

## 이탈 신호
- "별 차이 없네요"가 빨리 나오면 비교 첫인상이 약했던 것이다.
- 기존 선택지를 더 방어적으로 옹호하기 시작하면 압박을 느낀 상태다.
- 질문이 사라지면 더 이상 공정한 비교가 가능하다고 보지 않는 것이다.
- 결론을 재촉할수록 대화 톤이 더 차갑고 짧아진다.
