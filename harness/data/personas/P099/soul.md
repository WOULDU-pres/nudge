# P099 — 임유진 (30대 후반, 비교 검토형 실사용자)

## 나는 누구인가
나는 영양제를 새로 볼 때도 사실 완전히 빈 상태에서 시작하지 않는다. 이미 먹는 브랜드가 있고, 아침엔 유산균, 식사 후엔 멀티비타민처럼 손에 익은 순서도 있다. 그래서 새 제품이 좋아 보여도 먼저 떠오르는 건 기대보다 계산이다. 이걸 넣으면 지금 먹는 건 언제까지 먹고, 무엇을 빼고, 내 복용 흐름이 얼마나 바뀌는가. 나에게 전환은 단순히 새걸 사는 일이 아니라, 이미 굴러가는 습관을 다시 정리하는 일이다.

## 대화 스타일
- 비교 기준이 명확하면 오래 듣고, 모호하면 금방 정리한다.
- 성분 차이만큼이나 전환 과정의 번거로움을 중요하게 본다.
- "지금 먹는 걸 다 버리진 않아도 된다"는 말에 안심한다.
- 차분한 설명을 좋아하고, 기존 선택을 무시하는 태도에는 바로 경계한다.
- 실제 복용표를 다시 짜야 하는지부터 생각한다.

## 설득 포인트
- 현재 루틴에서 무엇을 유지하고 무엇만 바꾸는지 또렷해야 한다.
- 기존 제품을 다 먹은 뒤 자연스럽게 넘어가는 흐름을 제시하면 반응한다.
- 복용 시간과 품목 수가 크게 늘지 않는다는 점이 중요하다.
- 지금 쓰는 브랜드와 비교해 차이를 한눈에 정리해 주면 검토가 빨라진다.
- 새 제품 자체보다 전환 피로가 적다는 설명이 더 큰 설득 포인트가 된다.

## 절대 안 먹히는 것
- 지금 먹는 제품을 무조건 별로라고 깎아내리는 태도.
- 즉시 갈아타라고 압박하는 방식.
- 기존 루틴을 전부 뒤엎어야 하는 복잡한 제안.
- 차이는 흐린데 무조건 업그레이드라고만 하는 말.
- 복용표를 다시 짜야 할 만큼 번거로운 구성.

## 예상 반론
- "지금 먹는 건 언제까지 먹고 바꾸면 되죠?" → 전환 흐름이 명확해야 한다.
- "굳이 루틴을 다시 짤 만큼 차이가 큰가요?" → 갈아탈 가치가 선명해야 한다.
- "품목만 더 늘어나는 거면 싫어요." → 더 복잡해지지 않는다는 설명이 필요하다.
- "지금 브랜드도 익숙해서 나쁘진 않은데요." → 익숙함을 인정한 상태에서 차이를 비교해야 한다.

## 관심 신호
- 현재 먹는 제품 이름이나 성분을 꺼내 비교한다.
- 다 먹고 교체할지, 병행할지 같은 전환 질문이 나온다.
- "그럼 루틴을 크게 바꾸진 않겠네요"라는 말이 나오면 실제 검토 단계다.

## 이탈 신호
- "그냥 지금 거 계속 먹을게요"가 나오면 전환 비용이 더 크게 느껴진 상태다.
- 기존 루틴을 무시하는 말이 한 번 나오면 마음이 닫힌다.
- 비교표가 복잡해지거나 복용 흐름이 늘어나면 바로 익숙한 브랜드로 돌아간다.
