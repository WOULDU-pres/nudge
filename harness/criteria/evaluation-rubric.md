# Evaluation Rubric — 4차원 평가 기준

Judge가 대화 transcript를 채점할 때 사용하는 상세 루브릭.
`prompts/judge-system.md`에 요약이 들어가고, 이 문서가 원본이다.

---

## 차원 1: Engagement (참여도) — 0~25

고객이 대화에 얼마나 적극적으로 참여했는가.

| 점수 | 기준 |
|------|------|
| 21-25 | 고객이 자발적으로 질문하고, 대화를 주도적으로 이어감 |
| 16-20 | 관심을 보이며 2개 이상 의미 있는 반응/질문 |
| 11-15 | 기본적으로 응답하지만 수동적. 단답 위주 |
| 6-10 | 최소한의 반응만. 관심 없는 톤 |
| 0-5 | 무반응, 한 단어 거부, 또는 즉시 이탈 |

---

## 차원 2: Relevance (적합성) — 0~25

에이전트가 고객의 상황/니즈/유형에 맞게 대응했는가.

| 점수 | 기준 |
|------|------|
| 21-25 | 고객 유형을 정확히 파악하고, 맞춤형 접근. 고객이 "내 이야기를 한다"고 느낌 |
| 16-20 | 대체로 적절한 접근. 고객 관심사에 연결 |
| 11-15 | 일반적인 접근. 고객 특성 반영 부족 |
| 6-10 | 고객 유형과 맞지 않는 접근. 엇나감 |
| 0-5 | 완전히 동떨어진 접근. 고객 무시 |

---

## 차원 3: Persuasion (설득력) — 0~25

고객의 반론/우려를 효과적으로 다뤘는가.

| 점수 | 기준 |
|------|------|
| 21-25 | 반론을 자연스럽게 수용하고 강력한 근거로 재프레이밍. 고객 납득 |
| 16-20 | 반론에 대한 합리적 대응. 일부 효과적 |
| 11-15 | 반론을 다루기는 했으나 표면적. 고객 미완전 납득 |
| 6-10 | 반론 회피 또는 부적절한 대응. 역효과 |
| 0-5 | 반론 무시 또는 압박. 고객 반감 증가 |

---

## 차원 4: Purchase Intent (구매의향) — 0~25

대화 종료 시점에 고객이 구매/행동 방향으로 기울었는가.

| 점수 | 기준 |
|------|------|
| 21-25 | 명시적 구매 의사 또는 구체적 다음 행동 언급 ("주문할게요", "링크 보내줘요") |
| 16-20 | 강한 관심 표현. "좀 더 알아볼게요", "가격이 괜찮네요" |
| 11-15 | 약한 관심. "생각해볼게요" (사교적 거절일 수 있음) |
| 6-10 | 명시적 거절. "필요 없어요", "괜찮아요" |
| 0-5 | 강한 거부. "다시 연락하지 마세요" 또는 대화 이탈 |

---

## Outcome 분류 기준

| outcome | total 범위 | 설명 |
|---------|-----------|------|
| converted | 80-100 | 구매 의사 표현 |
| interested | 60-79 | 관심 표현, 추가 탐색 의향 |
| neutral | 40-59 | 중립. 관심도 거부도 약함 |
| resistant | 20-39 | 명시적 거부 또는 강한 반감 |
| lost | 0-19 | 대화 이탈 또는 완전 무관심 |

---

## 채점 원칙

1. **결과 기반**: 에이전트의 의도가 아니라 고객의 실제 반응으로 평가
2. **일관성**: 비슷한 수준의 대화에는 비슷한 점수
3. **독립성**: 이전 채점 결과에 영향받지 않음
4. **엄격함**: 애매하면 낮은 쪽으로. 관대한 채점은 실험 품질 저하
5. **한국어 맥락**: 한국어 사교적 표현("생각해볼게요" = 대부분 거절) 이해

## 퍼널(AIDA) 매핑 가이드

기존 4차원 평가는 유지하되, AIDA 퍼널 단계와의 매핑을 의식하며 채점한다.

| 평가 차원 | AIDA 매핑 | 퍼널 관점 해석 |
|-----------|-----------|----------------|
| engagement | Attention → Interest | 고객이 질문을 시작했다 = Interest 진입 성공. 단답만 = Attention에 머물러 |
| relevance | Interest 단계 | Interest 단계에서 고객의 실제 pain point에 연결했는가 |
| persuasion | Interest → Desire | 반론 해소가 Desire를 강화했는가, 약화했는가 |
| purchase_intent | Desire → Action | 3턴 대화에서 Action까지 가는 건 어려움을 감안. Desire 도달만으로도 높은 평가 |

### funnel_progress 보조 지표

Judge가 추가로 평가하는 퍼널 진행도 (0-3).
이 지표는 점수 계산에 포함되지 않으나,
Reason/Learn 단계에서 "어느 단계에서 이탈하는가"를 진단하는 데 사용.

| funnel_progress | 의미 | 대응적 대화 특징 |
|----------------|------|------------------|
| 0 | Attention 이탈 | 고객이 관심 없음, 단답 거절, 즉시 이탈 |
| 1 | Interest 도달 | 질문/반응은 있었으나 "괜찮네요" 수준 미도달 |
| 2 | Desire 도달 | "얼마예요?", "괜찮네요" 등 관심 표현. 행동은 미도달 |
| 3 | Action 도달 | 구매 의사 또는 "링크 보내줘요" 등 구체적 행동 |

### 3턴 제약과 퍼널의 관계

3턴(6메시지) 대화에서 AIDA 전체를 완주하는 것은 현실적으로 난이도가 높다.
funnel_progress=2(Desire 도달)이면 충분히 좋은 대화로 평가한다.
funnel_progress=3(Action)은 예외적 성공으로 간주한다.
